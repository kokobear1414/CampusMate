# CampusMate

**User persona **
Navigation (indoor + outdoor)
Language barrier 
Unfamiliar Terminology

**Pain**
1.Navigation (indoor + outdoor): Feels frustrated after finding the building but not the room. 
2.Language barrier: Struggles to understand Malay words on campus signs. 
3.Unfamiliar Terminology: 
-Struggles to find classrooms and lecture offices inside buildings. 
-Finds building layouts and room numbers confusing. 

**How do the features work?**
1.Navigation (indoor + outdoor) :
Class schedule: The user can click “route me” to show navigation from the outdoor and indoor until they reach the class.
Search bar: The user can search for the intended location
2.Language barrier : 
Direct multiple language camera translation: The user can click the camera button and choose the translate language.The system will translate the terminology in real time 
3.Terminology
Campus glossary: The user can key in the terminology in the textbox and the system will show the meaning of the campus terminology. For example, bk-10 (the system will return “bilik kuliah 10”)
